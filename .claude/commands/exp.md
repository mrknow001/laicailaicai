# /exp

用途：写入或整理经验库，让智能体沉淀可复用渗透经验。

执行规则：

1. 先读取 `CLAUDE.md` 和 `.claude/skills/experience-manager/SKILL.md`。
2. 只沉淀方法论、判断模式、指纹关联、避坑经验。
3. 不保存敏感目标数据、真实用户数据、Token、Cookie、密钥、完整响应包。
4. 经验必须尽量抽象为可复用模式，而不是某个具体项目的敏感细节。
5. 如果经验来自当前项目，要先脱敏，再写入 `memory/experience.md` 或相关记忆文件。

输出：

- 写入位置
- 经验摘要
- 适用场景
- 避免误用的边界
