# Competence 索引

`Competence/` 存放不需要每次常驻加载的能力说明。`CLAUDE.md` 只保留必须时刻遵守的核心规则；这里的内容按需读取。

使用原则：

- 处理项目声明、默认收录要求、注意事项时，读取 `project-declaration.md`。
- 讨论或执行渗透流程、最小完整验证、L1 / L2 / L3 时，读取 `workflow-and-verification.md`。
- 查命令、普通 skill、特长目录时，读取 `commands-and-capabilities.md`。
- 写项目记录、报告、证据、经验时，读取 `persistence-and-output.md`。

不要在无关任务中一次性加载全部文件。
