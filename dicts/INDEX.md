# 字典索引

字典目录用于存放低风险、授权范围内使用的路径、参数、业务关键字等字典。

建议字典：

| 文件名 | 说明 |
| --- | --- |
| `1-dir-framework.txt` | 常见框架、组件、API 文档、actuator、druid 等路径 |
| `1-dir-fuzz.txt` | 大型路径和词根字典，慎用 |
| `1-dir-system.txt` | 常见业务系统路径 |
| `dir-fuzz-top1000.txt` | 轻量路径字典，优先用于低风险验证 |
| `orderby.txt` | 排序字段字典 |
| `password-top-x.txt` | 常见密码，仅用于风险研判和报告说明，禁止爆破真实账号 |
| `super-verify.txt` | 常见验证码样例，仅用于风险研判，禁止批量验证 |
| `test-phone.txt` | 测试手机号样例 |

默认原则：

- 优先小字典、低速、可熔断。
- 字典必须服务于项目收录要求和高价值漏洞假设。
- 禁止用于撞库、密码喷洒、真实账号爆破、破坏性 fuzz。
